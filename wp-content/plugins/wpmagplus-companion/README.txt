== Description ==
WPMagPlus is generic plugin to companion plugin that will help import demo content for your webiste.

This plugin imports content, widget and customzier. An extended plugin can be developed in order to import your own demo data.



== Installation ==

= Uploading in WordPress Dashboard =

1. Go to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select wpmagplus-companion.zip from your computer.
4. Click 'Install Now'
5. Activate the plugin through the 'Plugins' menu in WordPress
